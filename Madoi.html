My Contribution

This description is the one from Wisp Renderer and serves as a place holder, A correct one will be writen Sunday July 7th. I started working on this project since I wanted to improve my skill as a graphics programmer and was looking forward to a tech focused project. In the initial stages of the project I've been working on implementing the basics of a PBR pipeline into our deferred rasterized renderer. During this time I've also worked on some general engine maintenance to improve the quality of life during future development. Later on I started working on implementing Raytraced Ambient Occlusion during which my understanding of both DirectX 12 increased significantly. In the final stages of the project I've implemented crash logging and switched back and forth iterating on the Ambient Occlusion render task and creating an editor for the graphics settings to tweak the render tasks during run time.
